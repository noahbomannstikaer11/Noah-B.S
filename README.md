# 💼 Business Tycoon 3D - Money Making Simulator

## 🎮 Spilkoncept
Et 3D-spil hvor du starter med $1000 og skal bygge en forretningsimperium op. Køb små forretninger, opgradér dem, og konkurrér mod AI-virksomheder.

## ✨ Features
- ✅ Start med $1000
- ✅ Køb små forretninger (kiosk, butik, osv.)
- ✅ Upgrades for at øge profit
- ✅ Konkurrence med AI-virksomheder
- ✅ Ekspandér til større bygninger/imperium
- ✅ 3D-miljø som du bevæger dig rundt i

## 🚀 Kom i gang
1. Åbn projektet i Unity 2022.3 LTS eller nyere
2. Åbn `MainScene` i Scenes mappen
3. Tryk Play og start med at bygge dit imperium!
